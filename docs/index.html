src/RandomForestBase.js

Create a new base random forest for a classifier or regression model.

new constructor(options: object, model: object)
Parameters
options (object)
Name Description
options.maxFeatures (number | String)? the number of features used on each estimator. if is an integer it selects maxFeatures elements over the sample features. if is a float between (0, 1), it takes the percentage of features.
options.replacement boolean? use replacement over the sample features.
options.seed number? seed for feature and samples selection, must be a 32-bit integer.
options.nEstimators number? number of estimator to use.
options.treeOptions object? options for the tree classifier, see [ ml-cart ] https://mljs.github.io/decision-tree-cart/
options.isClassifier boolean? boolean to check if is a classifier or regression model (used by subclasses).
model (object) for load purposes.
